简单工厂模式 :
    在old的版本中,没有工厂,所有每个商家(OrderPizza)都需要自己去判断要去哪一个生产线(就是不同的pizza类)上订购,
    新增了PizzaFactory类后,就相当于有一个工厂将所有的生产线整合了,这下不同的商家就不用自己判断去哪个生产线上订购了,
    直接去工厂订购,在将得到的pizza出售就可以了(实际上pizza中的方法也应该在factory中做,才是最便于理解的)

静态工厂模式 :
    其实就是简单工厂模式的静态方法版本

当有新的pizza种类时,商店只需要type发给工厂,而不再需要自己去寻找不同的生产线了,新增/删除某个种类的pizza,也不用去
多个商店类中修改了
